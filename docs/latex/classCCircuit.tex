Checks the validity of the circuit. 

Check if the circuit is valid.

This function checks the validity of the circuit based on the following conditions\+:
\begin{DoxyEnumerate}
\item Every unit must be accessible from the feed.
\item Every unit must have a route forward to both of the outlet streams.
\item There should be no self-\/recycle.
\item The destination for both products from a unit should not be the same unit.
\item The destination id of each unit must be between 0 to num\+\_\+units + 1. The first destination must not be tailings, the second destination must not be concentrate.
\end{DoxyEnumerate}

\begin{DoxyReturn}{Returns}
True if the circuit is valid, false otherwise.
\end{DoxyReturn}
This function checks for various conditions for circuit validity, including unit accessibility from the feed, route forward to both of the outlet streams, no self-\/recycle, different destinations, and appropriate destination ID.
